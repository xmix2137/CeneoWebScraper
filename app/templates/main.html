Cel aplikacji

Celem aplikacji jest pobierania z serwisu Ceneo.pl opinii o produktach na podstawie ich indywidualnych kodów oraz do wyświetlenia pobranych opinii i podstawowych statystyk o nich.

Opis działania aplikacji

Pobiera dane o produkcie z platformy ceneo.pl o podanym wcześniej kodzie i zapisuje je w postaci pliku JSON następnie umożliwia obróbkę tych danych.

Zastosowane biblioteki

  • Requests
  • BeautifulSoup4
  • Json
  • Os
  • Pandas
  • Numpy
  • Matplotlib
  • Flask
